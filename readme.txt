RBX Gallery can be used to show multiple images for a product you are selling.You can the image view style in options like JQ Zoom or Light Box.== Screenshots ==Coming Soon== Installation ==	=>RBX Gallery and Slider		1. 	Install & Activate the plugin.	2.  Go to RBX Gallery.	3.  Upload Images.	4. 	Go to your post/page.	5. In RBX Gallery Meta Box, click on Add rbx gallery images link and choose images from RBXGAllery tab for your post/page.	6. Add '[rbxgallery]' tag if you want to show gallery for post/page.	7. Add [rbxslider] tag if you want to show slider for post/page.	8. In order to show gallery or slider at particular place, simply Put `<?php echo do_shortcode("[rbxgallery]"); } ?>` for gallery and `<?php echo do_shortcode("[rbxslider]"); } ?>` for slider in your php code.	That's it ... Have fun== Changelog ==1.0 is launchedNew Setting Has been added for RBX Gallery and now has functionality to add a slider to your site.1.1 Issues fixed2.0 RBX Gallery with zoomer and  RBX Slider2.1 Bugs fixed3.0 Bugs Fixed and new improvements3.1 Bugs Fixed== Frequently Asked Questions ===> When writing a page/post, you can use the follow tags:	Gallery :- [rbxgallery]Slider :- [rbxslider]=> RBX Gallery Will work for any custom type post.=> New image view styles like ZQ Zoom Or Light Box in RBX Gallery.=> RBX Slider available in six different themes.
